# aula-de-segunda-feira

Este repositório contém um programa em Portugol que demonstra o uso de variáveis e tipos de dados.

## Arquivo

- `programa.por` - Programa principal em Portugol

## Descrição do Programa

O programa demonstra:
- Declaração de variáveis (cadeia, caracter, inteiro, real, logico)
- Inicialização de variáveis
- Atribuição de valores
- Saída de dados com `escreva`

## Variáveis utilizadas:
- `nome` (cadeia): "Paysandu"
- `primeira_letra` (caracter): 'H'
- `idade` (inteiro): -99
- `altura` (real): 5.50
- `data_de_nascimento` (cadeia): "30/03/1974"
- `presenca` (logico): verdadeiro
